# Rustfolio Tutorial – Financial Planning Module

> Audience: Long-term investors planning retirement or major life goals  
> Goal: Estimate future financial outcomes based on income, expenses, assets, and contributions.

---

# 🏦 What Is the Financial Planning Module?

This feature answers:

> "Will I reach my financial goals given my current situation?"

It combines:

- Portfolio projections  
- Income & expense tracking  
- Asset & liability overview  
- Retirement simulations  
- What-if scenario modeling  

This turns Rustfolio from a portfolio tracker into a life-planning tool.

---

# 📊 Retirement Projection Dashboard

At the top you see:

- Current Portfolio Value  
- Projected Retirement Value  
- Estimated Monthly Income  
- Shortfall / Surplus Indicator  

A graph shows:

- Current plan  
- Monthly contribution scenario  
- Surplus needed scenario  
- Custom scenario  

This visualizes your financial trajectory over time.

---

# 🔄 What-If Scenarios

You can simulate:

- Increasing monthly contributions  
- Reducing contributions  
- Changing retirement age  
- Adjusting return assumptions  

Example:
Scenario A → +$450 monthly  
Projected value increases  
Income gap decreases  

This helps test:

> "What happens if I invest more?"

---

# 📋 Financial Planning Survey

The module walks you through structured steps.

Steps shown:

1. Personal Details  
2. Spouse / Partner  
3. Income & Expenses  
4. Assets  
5. Liabilities  
6. Goals  
7. Risk Profile  

Each step builds your financial profile.

---

# 👥 Spouse / Partner Section

You can:

- Include spouse income  
- Add partner assets  
- Combine retirement planning  

This supports household-level planning.

---

# 💰 Income Tracking

You can add:

- Salary  
- Dividends  
- Rental Income  
- Other recurring income  

Each entry includes:

- Monthly amount  
- Owner (You / Partner)  
- Recurring frequency  

This determines retirement cash flow projections.

---

# 🧾 Expense Tracking

Expenses are categorized:

- Housing  
- Utilities  
- Groceries  
- Education  
- Transportation  
- Entertainment  
- Healthcare  
- Insurance  

Adding realistic expenses is critical for accurate planning.

---

# 🏦 Assets Section

You can manually add:

- Investment accounts  
- Bank accounts  
- Real estate  
- Retirement plans (RRSP, TFSA, RESP, etc.)  

Or link portfolio accounts automatically.

Each asset shows:

- Owner  
- Current value  
- Category  

---

# 💳 Liabilities Section

You can add:

- Mortgage  
- Loans  
- Credit cards  

This impacts:

Net worth  
Debt burden  
Retirement readiness  

---

# 🎯 Goals Section

Example shown:

Retirement Goal  
$1,500,000 target  

Progress bar shows:

Current progress toward target.

This helps answer:

> "Am I on track?"

---

# 📈 Retirement Income Estimation

The system estimates:

- Monthly retirement income  
- Portfolio drawdown  
- Surplus / deficit  

If deficit appears:

You may need:

- Higher contributions  
- Later retirement age  
- Lower expense assumptions

# 🧠 How Beginners Should Use This Module

Step 1:
Enter accurate income and expenses.

Step 2:
Add all assets and liabilities.

Step 3:
Set realistic retirement goal.

Step 4:
Run scenarios.

Step 5:
Adjust contributions if needed.
