# Rustfolio Tutorial – Analytics Page

> Audience: Beginners who want to understand performance trends and future projections  
> Goal: Learn how to interpret portfolio performance charts and forecasting models without trading experience.

---

# 📊 What Is the Analytics Page?

The **Analytics** section answers two big questions:

1. 📈 How has my portfolio performed over time?
2. 🔮 Where might it go in the future?

It has two main tabs:

- **Portfolio Performance**
- **Portfolio Value Forecast**

This is where numbers turn into trends and projections.

---

# 📈 Tab 1 – Portfolio Performance

This tab shows a performance chart with selectable overlays.

### Controls Available

- SMA (Simple Moving Average)
- Trendline
- Bollinger Bands
- Time filters (1M, 3M, 6M, 1Y, etc.)
- Date range selector

---

## 📉 What the Performance Chart Shows

The chart displays:

- Portfolio value over time
- Trend direction
- Smoothing indicators (if selected)

### Beginner Explanation

Think of this like your investment journey timeline.

If the line slopes upward:
You're growing wealth.

If the line drops sharply:
You experienced a drawdown.

---

## ⚠️ Limited Historical Data Warning

The video shows a notice:

> Technical indicators require more historical data.

This means:
If your portfolio history is short, advanced analytics may not be fully reliable yet.

More data = more accurate trend analysis.

---

# 📊 Portfolio Metrics Panel (Right Side)

This section summarizes:

- Total Value
- Total Cost
- Total Gain
- Return %

Below that:

### Risk Analysis Metrics

- Volatility
- Sharpe Ratio
- Beta
- Max Drawdown
- Risk Score

These help you measure performance quality — not just raw returns.

---

# 📦 Allocation Section

Shows percentage allocation by:

- Individual holdings
- Asset type
- Sector (depending on configuration)

This connects performance with structure.

If performance drops:
Check if one heavily weighted asset caused it.

---

# 🔮 Tab 2 – Portfolio Value Forecast

This is the predictive section.

It estimates potential future value using models.

---

## Forecast Overview

Displays:

- Current value
- Forecasted value
- Expected return
- Confidence bands (shaded area)

The shaded region represents uncertainty.

Wider band = less certainty.

---

# 🧠 Forecasting Methods

Three methods shown:

### 1️⃣ Ensemble
Combines multiple models.

Most balanced approach.

Recommended for beginners.

---

### 2️⃣ Exponential
Assumes consistent growth trend.

Good when portfolio has steady upward movement.

---

### 3️⃣ Moving Average
Uses historical smoothing.

More conservative and reactive.

---

# 📅 Time Range Selector

Example shown:

- 1 Year
- 3 Months

Changing the time range adjusts:

- Projection length
- Confidence spread

Shorter range = tighter forecast  
Longer range = wider uncertainty

---

# 📊 Confidence Band Explained

The blue shaded area represents:

> 95% confidence range

This means:
The portfolio is statistically expected to fall within this range 95% of the time.

Important:
Forecasts are probabilities, not guarantees.

---

# ⚠️ Important Considerations Panel

The video shows warnings such as:

- Forecast based on benchmark assumptions
- Fixed income approximations
- Market similarity assumptions

These remind users:

Forecasts are mathematical models — not predictions of specific events.

---

# 🧠 How Beginners Should Use Forecasting

Use forecasts to:

- Understand long-term trajectory
- Compare conservative vs aggressive outlook
- Visualize compounding over time

Do NOT use forecasts to:

- Time the market
- Make short-term trades
- Assume guaranteed returns

---

# 📉 Max Drawdown Reminder

Even if forecast shows growth,
markets still experience downturns.

Max Drawdown metric shows historical worst decline.

Combine:
Forecast + Risk Metrics

To understand both upside and downside.

---

# ⚠️ Common Beginner Mistakes

❌ Treating forecast as certainty  
❌ Ignoring confidence bands  
❌ Comparing short-term projections  
❌ Focusing only on return %  

---

# ✅ What This Page Helps You Answer

- Is my portfolio trending upward?
- How volatile is my growth?
- What's a realistic projection?
- What could my portfolio look like in 1 year?
- How uncertain is that estimate?

---

# 📌 Key Takeaway

Performance tells you:
> "What happened?"

Forecast tells you:
> "What might happen — statistically."

Use both together for smarter long-term planning.
