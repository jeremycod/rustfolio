# Rustfolio Tutorial – Downside Risk Metrics

> Audience: Beginners who want to focus on harmful volatility  
> Goal: Measure only harmful volatility (not upside swings).

---

# 📉 What Is Downside Risk?

Downside risk measures only negative volatility.

It answers:

> "How much does my portfolio drop — ignoring gains?"

This is more relevant than total volatility for most investors.

---

# 📊 Downside Deviation

Example:

```
30.21%
```

Measures volatility only when returns fall below a minimum threshold.

Unlike standard deviation, it ignores upside movement.

---

# 📈 Why Downside Deviation Matters

Investors care about:

Losses — not gains.

If a stock:
- Gains 20% one month
- Loses 15% next month

Standard deviation treats both equally.  
Downside deviation only penalizes the loss.

---

# 📊 Sortino Ratio

Example:

```
2.15
```

Similar to Sharpe Ratio, but penalizes only downside risk.

Formula (simplified):

```
Sortino = Return / Downside Deviation
```

Higher = better downside protection.

---

# 🧩 Sortino vs Sharpe

Sharpe Ratio:
Penalizes all volatility

Sortino Ratio:
Penalizes only downside volatility

Sortino is often more realistic than Sharpe for long-term investors.

---

# 📌 How To Interpret Sortino

- Below 1 = poor downside-adjusted return
- 1-2 = acceptable
- Above 2 = good
- Above 3 = excellent

A Sortino of 2.15 means:
You're getting good returns relative to downside risk.

---

# 🧠 Why This Matters

Two portfolios may have same Sharpe ratio,
but different Sortino ratios.

The one with higher Sortino:
Has better downside protection.

---

# 📌 How Beginners Should Use This Page

Step 1:
Check Downside Deviation

Step 2:
Compare to standard deviation

Step 3:
Review Sortino Ratio

Step 4:
Prefer positions with higher Sortino

---

# ⚠️ Common Beginner Mistakes

❌ Ignoring downside metrics  
❌ Focusing only on Sharpe Ratio  
❌ Not distinguishing upside vs downside volatility  

---

# ✅ What This Page Helps You Answer

- How much downside volatility do I have?
- Am I compensated for downside risk?
- Which positions have better downside protection?
- Should I prefer Sortino over Sharpe?

---

# 📌 Final Takeaway

Sharpe Ratio:
> "Return per unit of total risk"

Sortino Ratio:
> "Return per unit of downside risk"

For most investors, Sortino is more meaningful.
