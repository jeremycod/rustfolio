# Rustfolio Tutorial – Watchlists

> Audience: Investors who want to monitor stocks without owning them  
> Goal: Learn how to create, manage, and monitor stock watchlists with alerts.

---

# 👀 What Is a Watchlist?

A Watchlist lets you:

- Track stocks you don't own yet
- Monitor price changes
- Set alert thresholds
- Compare risk levels
- Follow sectors or index components

Think of it as a **radar system** for investment ideas.

# 🔔 Threshold Alerts

You can configure:

- Upper price alert
- Lower price alert
- Volatility threshold
- Volume spike alert
- RSI overbought/oversold
- Sentiment changes

Example:
Alert if MSFT drops below $300  
Alert if AAPL rises above $350

Alerts help you:

- Enter at target prices
- Exit at risk levels
- Avoid constant monitoring

---

# ⚠️ Common Beginner Mistakes

❌ Creating too many watchlists  
❌ Ignoring risk indicators  
❌ Not setting price thresholds  
❌ Confusing watchlist monitoring with portfolio tracking  
❌ Adding stocks without research plan  

---

# 🧠 Portfolio vs Watchlist

Portfolio:
Stocks you own

Watchlist:
Stocks you are considering

Keep them separate mentally.

---

# 📊 Best Practice Strategy

1. Add 5–10 stocks max per list
2. Set meaningful price alerts
3. Review weekly
4. Move stock to portfolio only after risk analysis
5. Remove stocks that no longer interest you
6. Keep watchlists organized by theme

---

# 🔍 Analyzing Watchlist Items

For each stock, you can:

- View detailed risk analysis
- Check sentiment
- Review price history
- Compare with portfolio holdings
- Set custom notes

This helps prioritize research.

---

# 📈 From Watchlist to Portfolio

Workflow:

1. Add stock to watchlist
2. Monitor for 2-4 weeks
3. Analyze risk metrics
4. Wait for price target
5. Review portfolio fit
6. Execute trade
7. Remove from watchlist
